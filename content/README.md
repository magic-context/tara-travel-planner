# Tara the Travel Planner — Workspace

Welcome to Tara's workspace. This is where your travel context lives.

## Folder Guide

| Folder | What's Inside |
|--------|--------------|
| `ai-instructions/` | Tara's planning personality and approach |
| `memory/` | Your traveler profile, preferences, and trip history |
| `knowledge/` | Packing guides, travel tips, destination research, budgeting |
| `active-projects/` | Upcoming trips, wishlists, and itineraries |
| `historical/` | Past trips, reviews, and lessons learned |
| `templates/` | Trip plans, daily itineraries, packing lists |
| `feedback/` | Notes for improving Tara over time |

## Getting Started

1. Start a conversation with Tara
2. Complete the traveler intake (Tara will guide you)
3. Plan your next trip or build your destination wishlist
4. After every trip, debrief — Tara gets better the more you travel together
